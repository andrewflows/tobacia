Bug-loop · auto PR · human approval gate

Every app has a bug button. Tap it. Your report opens a GitHub issue. A Claude agent running on my Tailscale-secured home server polls every night, opens fix branches as PRs, and waits for me to approve and ship. You get a reply. The loop closes — and the API key never leaves my infra.

1 · Report
Tap "Report bug" inside any app. Pre-filled issue.
2 · Auto-fix
Nightly home-server cron wakes Claude. Fix branch + PR.
3 · Ship
I review, merge, deploy. Reporter notified.
Why a personal app store?

Off-the-shelf apps are best-of-breed for broad use cases. They’re bad at niche, idiosyncratic, “only-I-care” workflows. AI coding agents collapse the cost of building niche software to nearly zero, which means individual operators can now own a full app store of tools tuned to their exact way of working.

This is mine. The apps in it solve problems I’ve hit personally over 17 years in industrial supply chain — the kind of problems where a generic SaaS tool gets you 60% of the way and the last 40% is friction every single day.

No compromises. No committee design. Built end-to-end by one operator + AI agents.
